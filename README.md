# Welcome to SnapLink

A secure file sharing application with password protection and expiration dates.

## Getting Started

### Development

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To build the application for production:

```bash
npm run build
```

### Deployment

The application can be deployed to any static hosting service. Simply run the build command and deploy the contents of the `dist` directory.

## Features

- Secure file sharing
- Password protection
- Expiration dates
- User authentication
- Dashboard for managing shared files
